# Indeed & Trucking Jobs Scraper 

Автоматизированный инструмент для сбора вакансий для водителей траков в США.

### Функционал:
- **Indeed Scraper**: Обход защиты с помощью Selenium, имитация действий пользователя (клики), сбор зарплат, компаний и прямых ссылок.
- **Data Export**: Автоматическое сохранение результатов в CSV через Pandas.
- **Robustness**: Обработка динамической подгрузки данных.

### Как запустить:
1. Установите зависимости: `pip install -r requirements.txt`

2. Запустите скрипт: `python indeed.py`
